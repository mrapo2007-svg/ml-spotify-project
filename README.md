# Spotify Hit Prediction

## Description
This project predicts whether a song is a hit based on Spotify features.

## Dataset
Spotify 2023 dataset

## How to run
pip install -r requirements.txt

Run notebooks in order:
T1_EDA.ipynb → T2_Supervised.ipynb → T3_Unsupervised.ipynb → T4_Ensemble.ipynb

## Results
- Logistic Regression: 0.75 accuracy
- Decision Tree: 0.80 accuracy
- Random Forest: 0.85 accuracy
